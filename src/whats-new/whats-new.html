Beta version of Enlist, when it was still in development.

We released Enlist in its beta to test the basic functionality and to look for any flaws in the product. Our classmates went through it for some amount of time. Some bugs were reported, which were resolved to the best of our efforts before the official release (v1.0) of Enlist.

We really appreciate all our friends who helped in making our platform more flawless. âœ¨

The beta version included the functionality of shared tasks or keep it as a personal task. If any user wants to keep the task to himself then (s)he can add using the PERSONAL feature. So, that task won't appear in other people's lists. If any user wants to share the task to the whole team/class, then (s)he can add using the SHARED feature. Hence, it will automatically sync to everyone's list. Edit, done and delete operations on a particular task are individual for everyone and the changes made by that user won't occur in other people's list.
