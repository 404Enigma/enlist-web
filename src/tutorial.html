Update Button

If any user joins team late then (s)he can click this button to get the tasks which are already present in the database. After clicking this button, the tasks will be synced in his/her to-do list as well.

Add button

This button is used to add new tasks in the to-do list.

Edit button

This button is used to edit any task that is already present. Edit operations on a particular task are individual for everyone and the changes made by that user won't occur in other people's list.

Done button

This button is used to complete any task. Done operations on a particular task are individual for everyone and the changes made by that user won't occur in other people's list.

Delete button

  • This button is used to delete any task. Delete operations on a particular task are individual for everyone and the changes made by that user won't occur in other people's list.

  • After 15 days of the due date, the task will be automatically deleted.

Radio Button

  • Personal radio button can be used if the user wants to keep the task to himself. These tasks won't appear in other people's lists.

  • Shared radio button can be used wants to share the task to the whole team. These tasks will appear in other people's lists.
